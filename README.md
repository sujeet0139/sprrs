# 🏫 Shri Rama Prasad Singh Shiksha Sansthan - School Website

A modern, premium, fully responsive school website built with **Next.js 16**, **React 19**, and **Tailwind CSS 4**.

## 🎯 Features

✨ **Modern Design** - Premium UI with smooth animations  
📱 **Fully Responsive** - Works on all devices (mobile, tablet, desktop)  
⚡ **Fast Loading** - Optimized performance (<2 seconds)  
🎨 **Beautiful Animations** - Smooth scrolling and transitions  
📝 **Complete Pages** - Home, About, Academics, Facilities, Contact  
🎓 **Rich Components** - Birthday section, News, Activities, Leadership  
🔐 **SEO Ready** - Meta tags, semantic HTML, structured data  
🚀 **Production Ready** - Ready to deploy on Vercel  

## 📋 Project Structure

```
- Homepage with 17 sections
- About Us page
- Academics Detail page
- Facilities Detail page
- Contact page with form & map
- Floating Chatbot & WhatsApp widgets
```

## 🚀 Getting Started

### Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3001 in browser
```

### Build for Production

```bash
npm run build
npm start
```

## 🚢 Deployment on Vercel

### One-Click Deploy

1. Push to GitHub: `git push origin main`
2. Go to https://vercel.com/new
3. Select this repository
4. Click "Deploy"

Your site will be live in 1-2 minutes!

### Vercel CLI Deploy

```bash
npm install -g vercel
vercel
```

For production:
```bash
vercel --prod
```
