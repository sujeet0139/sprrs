## 🔧 HOW TO USE

### View Locally
```bash
npm install
npm run dev
# Open http://localhost:3001
```

### Deploy to Vercel
```bash
git push origin main
# Vercel auto-deploys on GitHub push
```

### Customize Colors
Edit `tailwind.config.js` or update color variables in components

### Add More Animations
Add CSS keyframes in `app/globals.css` and reference with `animate-*` classes
